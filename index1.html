Pikachu

Wavefinding Project for Blind and Visually Impaired People

About the Project

Welcome to the Pikachu Wavefinding Project. We are developing an innovative navigation system to assist blind and visually impaired individuals in navigating public spaces. Our goal is to provide an intuitive, audio-based system that allows users to confidently and independently navigate their surroundings.

The project is designed to enhance accessibility and independence for those with visual impairments by offering real-time guidance through voice commands and auditory cues. With this technology, users can navigate complex environments, such as parks, shopping malls, or public transportation systems, safely and efficiently.

Key Features

  • Real-time voice navigation with customizable voice options
  • Audio feedback for precise location and direction information
  • Haptic feedback integration for additional sensory input
  • Accessible interface designed specifically for visually impaired users
  • Adaptable for different environments (indoor/outdoor)
  • Obstacle detection and avoidance alerts
  • Landmark identification system

Project Draft

Our current draft includes the following components and milestones:

  • Phase 1 (Completed): Market research and user needs assessment
  • Phase 2 (In Progress): Prototype development with basic navigation features
  • Phase 3 (Upcoming): Advanced obstacle detection implementation
  • Phase 4 (Future): Integration with public transportation systems
  • Phase 5 (Future): Large-scale testing and user feedback collection

We are currently refining our algorithms for more accurate positioning and developing more natural voice guidance systems. Future updates will include indoor navigation capabilities and integration with existing accessibility tools.

Prototype

Our current prototype consists of a wearable device paired with a smartphone application. The system uses a combination of GPS, Bluetooth beacons, and computer vision to determine the user's location and provide navigation instructions.

Current prototype version 1.2

Prototype Features:

  • Wearable vibration feedback unit
  • Smartphone app with voice interface
  • Basic obstacle detection using ultrasonic sensors
  • Directional audio guidance system
  • Battery life of 8+ hours

We are currently testing the prototype with visually impaired volunteers in controlled environments. Initial feedback has been positive, with users particularly appreciating the clear voice instructions and intuitive vibration patterns for directional cues.
